# Real_Estate_Market_segmentation
This project was developed during my Financial Analyst Internship to automate the categorization of real estate investors and homebuyers for Parcl Co. Limited. Using a Python-based machine learning pipeline, the system analyzes client demographics and property preferences to generate actionable investment segments.
## Technologies Used
* *Language:* Python 3.10
* *Libraries:* Pandas, NumPy, Scikit-learn
* *Tools:* Command Prompt, Virtual Environments (.venv)

## How to Run
1. Ensure Python is installed.
2. Run pip install pandas scikit-learn
3. Execute the script: python Pipeline.py

## Output
The script generates final_buyer_segmentation.csv, which provides a categorized list of clients based on their financial and property-specific investment potential.
